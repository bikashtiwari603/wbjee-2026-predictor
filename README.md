# WBJEE 2026 Predictor

A static WBJEE college and branch predictor built from historical counselling data for 2022–2025.

## Features

- Rank-based college/branch prediction
- Category, quota, seat-type, and stream filters
- College-name and branch-name keyword search
- Works fully offline once the files are downloaded

## Files

- [wbjee_college_predictor.html](wbjee_college_predictor.html) — main app page
- [wbjee_predictor_data.js](wbjee_predictor_data.js) — generated predictor data
- [counselling data csv files](counselling%20data%20csv%20files) — source counselling CSV files

## GitHub Pages deployment

This project is ready to be published as a GitHub Pages site.

1. Push the repository to GitHub.
2. Open the repository settings.
3. Go to Pages and select the main branch with the root folder.
4. Your site will be available at:
   https://bikashtiwari603.github.io/wbjee-2026-predictor/

## Local preview

Open [wbjee_college_predictor.html](wbjee_college_predictor.html) in a browser, or serve the folder with any static file server.
